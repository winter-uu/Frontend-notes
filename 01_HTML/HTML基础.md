#### 一、什么是HTML
1、HTML ：hyper text markup language 超文本标记语言
超：超级（意思是多样化的）
文本：文字，图片，音频，视频
标记：是标签语言，也叫做标记语言
语言：作为程序员编写的代码，用于和计算机通讯的
2、另一层意思：超链接，页面可以跳转
#### 二、第一个HTML案例
1、代码：
```html
<html>
	<head>
			<title>这是标题</title>
	</head>
	<body>
			这是展示内容
	</body>
</html>
```
